🎮 SOBRE O PROJETO

QualJogador é uma aplicação interativa desenvolvida para fãs de futebol.

Através de perguntas relacionadas ao estilo de jogo, personalidade, características dentro de campo e preferências táticas, o sistema identifica qual jogador profissional possui um perfil mais próximo do usuário.

O projeto foi criado com foco em:

✓ Experiência do usuário
✓ Interface moderna
✓ Compatibilidade multiplataforma
✓ Navegação intuitiva
✓ Desenvolvimento com tecnologias atuais

🚀 INSTALAÇÃO

1. Clonar o repositório

git clone [https://github.com/MixuriM/qualJogador.git](https://github.com/MixuriM/qualJogador.git)

2. Entrar na pasta

cd qualJogador

3. Instalar dependências

npm install

4. Executar o projeto

npm start

═══════════════════════════════════════════════════════════════════════════════

📱 EXECUTAR NO ANDROID

npm run android

═══════════════════════════════════════════════════════════════════════════════

🍎 EXECUTAR NO IOS

npm run ios

═══════════════════════════════════════════════════════════════════════════════

🌐 EXECUTAR NA WEB

npm run web

═══════════════════════════════════════════════════════════════════════════════

🚀 DEPLOY

Gerar build web:

npm run build

Publicar no GitHub Pages:

npm run deploy
